Что выведет программа? Объяснить вывод программы. Объяснить внутреннее устройство интерфейсов и их отличие от пустых интерфейсов.

```go
package main

import (
	"fmt"
	"os"
)

func Foo() error {
	var err *os.PathError = nil
	return err
}

func main() {
	err := Foo()
	fmt.Println(err)
	fmt.Println(err == nil)
}
```

Ответ:
```
Функция Foo возвращает значение переменной err имеющую тип ссылки на структуру fs.PathError которая реализует интерфейс error.
Интерфейс состоит:

type iface struct {
    tab  *itab
    data unsafe.Pointer
}
Где tab это указатель на Interface Table или itable — структуру, которая хранит некоторые метаданные о типе и список
методов, используемых для удовлетворения интерфейса.
data — указывает на фактическую переменную с конкретным (статическим) типом.

err состоит из двух поинтеров, первый указывающий на itable для конкретной пары (статический тип type(fs.PathError),
интерфейс error), а data на копию оригинального значения fs.PathError{}. В err мы передали значение nil ссылки на тип
fs.PathError соответственно поле itab хранит метаданные о типе, а поле data равняется nil.
Интерфейс будет равняться nil когда оба поля интерфейса будут nil.

```
