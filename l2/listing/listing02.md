Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Программа выведет:
2
1

Функция fmt.Println вызывает функцию test() в которой переменная x именованная переменная возвращаемого значения.
В теле функции этой переменной присваивается значение x=1 и выполняется команда return. Значение 1 записано в стек. 
Так как функция отложенного вызова defer может к именованным возвращаемым параметрам окружающих функций и изменять их,
она инкрементирует переменную x и мы получаем значение x=2.

Во втором случае мы получаем значение x=1 потому что, **результат записан в стек, и (внимание) адрес области стека
с возвращаемым значением не равен области, отведённой для внутренней переменной x**.
Отложенная функция эту переменную обновит, но на результат это уже не повлияет, и программа выведет 1.

```
