Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Вывод программы:
[3 2 3]

Слайс это структура состоящая из ссылки на массив, значения количества элементов слайса len и ёмкости cap.
Слайс s при создании инициализируется данными, т.е. внутри слайса s создаётся массив с данными, поля len = 3, cap=3.
Данный слайс передаётся в функцию modifySlice, слайсы в golang передаются по ссылке, поэтому слайсы i и s внутри 
содержат ссылку на один и тот же массив. Когда внутри функции modifySlice элементу слайса под индексом 0 присваивается
новое значение, изменяется исходный слайс. Следующая операция append добавляет новый элемент в слайс, а у нашего слайса
len=cap, поэтому происходит копирование массива находящегосяв внутри слайса i в новый  увеличенный массив. После этого
изменения и добавления элементов в слайс i не приводят к изменениям в исходном слайсе, потому что, внутри слайсы s и i
ссылаются на разные массивы.

```
